# CCNA-Networking-Labs
This repository contains hands-on CCNA networking labs built using Cisco Packet Tracer.
The labs focus on core networking concepts including VLANs, Inter-VLAN routing, DHCP, NAT/PAT, ACLs, and routing protocols.

Technologies & Tools:

    Cisco Packet Tracer
    Cisco IOS CLI
    Networking protocols and services

Concepts Covered:

    Router CLI
    Static Routing
    Client Server Web Connection
    DNS
    VLAN
    Trunking
    Inter VLAN Routing
    Switch Virtual Interface
    Router On A Stick
    Dynamic Routing using RIP
    Dynamic Routing using OSPF
    Network Address Translation 
    EtherChannel
    VPN
    ACL
    Dynamic Host Configuration Protocol

Repository Purpose:

    This repository serves as a practical portfolio of networking labs while preparing for the CCNA certification and developing real-world network configuration skills.

How to Use:

    Download or clone the repository
    Download Cisco Packet Tracer
    Open .pkt files using Cisco Packet Tracer
    Explore device configurations and topology
