# CCNA-Networking-Labs
This repository contains my hands-on networking labs and projects created while learning CCNA concepts using Cisco Packet Tracer.

The goal of this repository is to document my learning journey through practical implementation of networking technologies, including:
    - Routing and Switching
    - VLANs and Inter-VLAN Routing
    - RIP and OSPF
    - DHCP, DNS, and NAT
    - ACLs and Network Security
    - IPv6
    - HSRP
    - VPN
    - Telnet and SSH
    - Wireless Networking
    - Enterprise Services and Administration

Each topic includes:
    - Lab file
    - Configuration commands
    - Topology image


I regularly share my progress and projects on LinkedIn as part of my networking learning journey.
