# CCNA-Networking-Labs

A collection of hands-on networking labs built using Cisco Packet Tracer to strengthen practical networking skills and reinforce CCNA concepts through real-world configurations.

## Overview

This repository documents my networking learning journey through practical implementation of routing, switching, network services, and security concepts. Each lab focuses on configuring and troubleshooting enterprise networking technologies using Cisco IOS CLI.

The repository serves as both a learning resource and a portfolio showcasing my progress toward CCNA-level networking proficiency.

---

## Tools & Technologies

* Cisco Packet Tracer
* Cisco IOS CLI
* Network Services & Security Protocols

---

## Topics Covered

### Network Fundamentals

* Router CLI Configuration
* Static Routing
* IPv6 Configuration

### VLAN & Switching

* VLAN Configuration
* Trunking
* VLAN Trunking Protocol (VTP)
* Inter-VLAN Routing (IVR)
* Switch Virtual Interface (SVI)
* Router-on-a-Stick (ROAS)
* EtherChannel

### Routing Protocols

* Routing Information Protocol (RIP)
* Open Shortest Path First (OSPF)

### Network Services

* DHCP Server Configuration
* DHCP Relay Agent
* DHCP Snooping
* DNS Configuration
* IIS Configuration

### Security & Administration

* Access Control Lists (ACL)
* Telnet
* SSH
* VPN Configuration
* Network Security Concepts

### Enterprise Networking

* NAT (Network Address Translation)
* PAT (Port Address Translation)
* HSRP (Hot Standby Router Protocol)
* Wireless Networking
